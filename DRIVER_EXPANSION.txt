ARCHITECTURE NEUTRALITY
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Before:
  menuconfig showed [x86_64] in title by default
  Architecture was pre-selected

After:
  menuconfig shows [] or [not selected]
  All architectures start as unselected (n)
  User selects architecture during make
  Configuration is architecture-neutral

Benefits:
  ✓ Single .config for all architectures
  ✓ Cross-compilation friendly
  ✓ CI/CD pipeline compatible
  ✓ Multi-arch build support
  ✓ No hardcoded assumptions

Usage:
  $ make menuconfig          # Configure (no arch selected)
  $ make ARCH=x86_64        # Build for x86_64
  $ make ARCH=arm64         # Build for ARM64
  $ make ARCH=riscv         # Build for RISC-V
